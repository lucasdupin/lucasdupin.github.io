TED South Experiment

South Experiment is a project created with the purpose of generating all the videos for 2014's TED Global conference.
I am proud of participating on two of them: Main Video (South!) and Mighty Spaces.
My role as a multimedia designer was mainly to create the visuals and animations based on the input data received.
You can read more below.

South!

The idea was to combine technology and art to generate an unique expression of South American culture.

The project is a combination of three components:
  • GeoTIFF terrain data from South America
  • Visuals programming using WebGL, Node.js and 2D Canvas
  • South American designs and illustrations
Using NASA's terrain data from several places in South America, I reconstructed the terrain geometry using GeoTIFF color depth.

The camera navigation was accomplished using sine and cosine waves, combined with vector math. On top of that some texture scroll helped to make the terrain feel alive.

For the last scene, Vector Buffers made the animation smooth.

Mighty Spaces

It started with a custom kite with tracking devices attached. We asked a kid to fly the kite, which is one of the most popular activities for kids to do in South America.

As he flew it, we were able to save the device's data in a text stream.
This stream was the base for plotting the kite's movement into a 3D scene, built with WebGL.

My part here was to write the socket stream receiver, translate the data into a mesh, animate and render it.
