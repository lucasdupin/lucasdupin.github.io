Read to a Child

Teaching how to read is something magical. Teaching how to read in a third world country makes you feel a better person.
This was the challenge, teching in a large scale, so, we developed a book companion app to make storytelling more pleasant for kids.

For this project we chose Corona/Lua as our programming environment to have a flexible UI framework (as if in a game) and tweaked it untill we got the best user experience.
Corona also let us develop crossplatform with ease, still enabling us to build native plugins.
